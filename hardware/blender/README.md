# Blender Hardware Models

This folder contains editable Blender source files for the rowing IMU housing and boat clamp prototypes.

| File | Purpose |
| --- | --- |
| `PleaseWaterResistant.blend` | Main water-resistant enclosure concept |
| `PleaseWaterResistant.blend1` | Blender Seat box with clamps, clamps should be mounted on the seat |
| `PleaseWaterResistantBoatcClamp.blend` | Boat clamp / might fit on footrest |
| `PleaseWaterResistantBoxOnly.blend` | Box-only enclosure model (7x7cm) |

GitHub stores these files as downloadable binary assets. For browser preview or 3D printing, export additional `.stl`, `.obj`, or `.glb` files from Blender.
